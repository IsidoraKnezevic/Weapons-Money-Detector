# Ingest Module for Autopsy

This is a data source ingest module for Autopsy that uses weapons_money_detector to detect pistols, bills, knives and credit cards in images.

## Installation

* In Autopsy open Tools menu then Python Plugins and copy this folder to the Autopsy folder for python modules. 
* Put exe file in folder `utils/dist`. Download at:
  * https://drive.google.com/file/d/1oeBAH9ux_yoUkbGjjX9dmzcHw8CHxLZ6/view?usp=sharing
